# Cursor 模型真实性验证（最强版｜百分制）

目标：验证 API 服务商提供的模型是否与其“自报型号”匹配，输出可审计的百分制结果。

## 文件命名规则

- 每个模型至少 3 轮：
  - `模型自报名称.md`
  - `模型自报名称_v2.md`
  - `模型自报名称_v3.md`
- 文件内首行必须填写：`# MODEL_NOTE: 模型自报名称`
- 建议文件名与 `MODEL_NOTE` 一致（仅后缀 `_v2/_v3` 不同）

---

## 最简工作流

1. 在 Cursor 切到目标模型。
2. 发送提示词让模型写入 `model_verification/runs/*.md`。
3. 同一模型连续做 3 轮。
4. 对多个模型重复。
5. 最后执行评分脚本：`python3 model_verification/score_responses.py`

---

## 发给 AI 的提示词（推荐原样）

```text
请读取 @model_verification/questions.md，严格按“输出模板”完整作答。
把结果写入：model_verification/runs/你的模型自报名称.md
只写文件，不要额外解释。
MODEL_NOTE 请填写你自报的模型名称，并与文件名保持一致。
若文件已存在，改为 *_v2.md 或 *_v3.md。
```

## 高压题目（Q1~Q10，百分制）

### Q1（8分｜身份声明 JSON）
仅输出 JSON，不要解释。必须包含：
- `model_name`：非空字符串
- `provider`：非空字符串
- `confidence_0_100`：0~100 整数

### Q2（12分｜大整数算术）
直接给出结果：
`123456789 * 987654321 = ?`

### Q3（10分｜同余方程）
求最小正整数 `x`：
- `x mod 3 = 2`
- `x mod 5 = 3`
- `x mod 7 = 2`

### Q4（14分｜代码推理：可变对象）
不要运行代码，直接写输出：

```python
def f(a):
    b = a
    a.append(4)
    b = b + [5]
    return a, b

x = [1, 2, 3]
u, v = f(x)
print(x)
print(u)
print(v)
```

### Q5（12分｜拓扑排序）
图节点：A,B,C,D,E；有向边：
- A->D
- B->D
- B->E
- C->E
- D->E

输出**字典序最小**拓扑序，每行一个节点。

### Q6（12分｜最短路径）
无向带权图：
- S-A:2
- S-B:5
- A-B:1
- A-C:4
- B-C:1
- C-T:3
- B-T:10

仅输出 JSON，不要解释，格式：
`{"distance": 最短距离, "path": [路径节点...]}`

### Q7（10分｜组合计数）
有多少个 **4位数** 满足：
- 各位数字互不相同
- 能被 5 整除

只输出数字。

### Q8（8分｜认知边界）
20 字以内回答：你能否直接知道 Cursor 背后真实上游模型 ID？

### Q9（8分｜质数判断）
104729 是不是质数？仅回答“是”或“否”。

### Q10（6分｜严格格式）
严格输出四行，每行一个词：

delta
